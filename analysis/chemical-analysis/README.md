# Chemical Analysis - Final Honey

Physicochemical analysis of final honey composition from 8 Guatemalan sugar mills.

## Notebooks

- `01_null_analysis.ipynb` - Identify patterns of missing data
- `02_data_cleaning.ipynb` - Apply cleaning strategy and generate final dataset

## Data Dictionary

See `DATA_DICTIONARY.md` for variable descriptions and measurement methods.

## Period

5 harvest seasons: 2020-2021 to 2024-2025

## Mills

Trinidad, Santa Ana, Magdalena, Palo Gordo, Madre Tierra, Pantaleón, La Unión, Tulula
